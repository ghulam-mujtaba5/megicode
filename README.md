## ğŸ Getting Started

### Quick Start
1. **Clone the repository:**
   ```sh
   git clone https://github.com/megicodes/megicode.git
   cd megicode
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Configure environment:**
   ```sh
   cp .env.example .env.local
   # Edit .env.local with your database credentials
   ```
4. **Setup database:**
   ```sh
   npm run db:migrate  # Run migrations
   npm run db:seed     # Populate test data
   ```
5. **Run the development server:**
   ```sh
   npm run dev
   ```
6. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Database Commands
- `npm run db:generate` - Generate migration files from schema changes
- `npm run db:migrate` - Apply migrations to database
- `npm run db:seed` - Seed database with sample data

### Testing & Quality
- `npm run test:e2e` - Run end-to-end tests with Playwright
- `npm run test:e2e:ui` - Open Playwright UI for debugging tests
- `npm run test:trace` - Capture full DevTools trace for analysis
- `npm run lint` - Run ESLint checks
- `npm run format` - Format code with Prettier

For comprehensive testing documentation, see [tests/E2E_TESTING_GUIDE.md](tests/E2E_TESTING_GUIDE.md).
