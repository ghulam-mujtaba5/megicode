TypeScript Error Fix Strategy:

1. JSON columns with custom types (ProcessDefinitionJson, AuditPayload, etc.) 
   - These are defined as .\<CustomType>() in schema
   - They should be passed as objects, NOT stringified
   - Drizzle handles JSON serialization internally

2. Fix pattern:
   - WRONG: json: JSON.stringify(obj)
   - RIGHT: json: obj (as CustomType)
   
   - WRONG: const parsed = JSON.parse(row.json)  
   - RIGHT: const parsed = row.json (already typed!)

3. Affected columns:
   - processDefinitions.json (ProcessDefinitionJson)
   - events.payloadJson (AuditPayload)  
   - integrations.config (IntegrationConfig)
   - meetingNotes.actionItems (MeetingActionItems)
   - templates fields (various)
   - projects.techStack (TechStack)
